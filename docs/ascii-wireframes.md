# Shinchon Montessori — Current ASCII Layouts (2025-10-29)

퍼블릭 웹과 학부모 포털의 현재 구현 상태를 ASCII 와이어프레임으로 요약했습니다.

---

## 1. Public Web (퍼블릭 사이트)

### 1.1 Home (`/`)
```
┌──────────────────────────────────────────────────────────────────────────────┐
| HERO (풀 너비, 그라데이션 배경)                                               |
|  Badge + Headline + Sub copy + [CTA: 입학 상담] [CTA: 프로그램]              |
|  통계 카드 3개                                                                |
├──────────────────────────────────────────────────────────────────────────────┤
| 교육 철학 카드 3개                                                            |
├──────────────────────────────────────────────────────────────────────────────┤
| 프로그램 소개(3열)                                                            |
├──────────────────────────────────────────────────────────────────────────────┤
| 하루 일과 타임라인                                                            |
├──────────────────────────────────────────────────────────────────────────────┤
| 입학 절차 스텝                                                                |
├──────────────────────────────────────────────────────────────────────────────┤
| 최신 소식 카드 3개                                                            |
├──────────────────────────────────────────────────────────────────────────────┤
| 방문/문의 CTA + 정보 카드                                                     |
└──────────────────────────────────────────────────────────────────────────────┘
```

### 1.2 News (`/news`, `/news/[slug]`)
```
List: 카테고리 탭 + 카드 (배지, 제목, 날짜, 요약, [자세히])
Detail: 헤더(제목, 카테고리, 작성 메타) + 본문(리치 텍스트) + 첨부 링크
```

### 1.3 Admissions / Programs 등 정적 페이지
```
Hero 배너 + 섹션 카드/리스트 (정적 텍스트 중심)
```

### 1.4 Meals (`/meals`)
```
Hero: Meal Program 배지 + 공개 식단 안내 카피

Section A — 달력
┌ Header: 제목 + 안내문 (전체 공개 식단만 노출) ──────────┐
| MealCalendar 컴포넌트 (조식/중식/간식 필터, 오늘 배지)  |
└────────────────────────────────────────────────────────┘

Section B — 다가오는 급식 카드
┌ Date Card ─────────────────────────────────────────────┐
| 날짜 헤더                                               |
| 식단 정보 (유형, 메뉴, 알레르기, 노트)                 |
| 사진 프리뷰 (최대 3장)                                  |
| 첨부 자료 링크                                          |
└────────────────────────────────────────────────────────┘

Section C — 영양사 공지
┌ Bulletin Card ─────────────────────────────────────────┐
| 카테고리 배지 · 게시일                                  |
| 제목                                                    |
| 본문 (pre-line)                                         |
└────────────────────────────────────────────────────────┘
```

### 1.5 Class News Preview (`/stories/class-news`)
```
Hero: 우리들 이야기 배지 + 안내 문구

Notice Panel
┌ Info Strip ─────────────────────────────────────────────┐
| 🔒 로그인 필요 안내 + [로그인하고 전체 보기] 버튼          |
└────────────────────────────────────────────────────────┘

Table (최근 12건)
┌ Header: No. / 반 / 제목 / 게시일 ───────────────────────┐
| 제목 클릭 시 로그인 모달 오픈, 요약은 2줄 표시          |
| Caption: 로그인 완료 시 학부모 포털로 이동              |
└────────────────────────────────────────────────────────┘

Info Cards
┌ Card 1: 학부모 포털 안내                                 ┐
| 캘린더 아이콘 + 설명                                     |
└────────────────────────────────────────────────────────┘
┌ Card 2: Privacy Badge + 안내 문구                        ┐
| 로그인 후 사진/첨부 공개 강조                           |
└────────────────────────────────────────────────────────┘

Login Modal
┌ Overlay: 반투명 배경 + 카드 ──────────────────────────┐
| 헤더: 로그인 후 자세히 확인                            |
| 본문: 이메일/비밀번호 입력 폼 + 오류/안내 메시지        |
| 하단: 회원가입 링크                                    |
└────────────────────────────────────────────────────────┘
```

## 2. Parent Portal (학부모 포털)

### 2.1 Dashboard (`/parents`)
```
Hero 인사말 + Admin 콘솔 이동 버튼(관리자일 때)

섹션 A: 등록된 자녀 카드 (반/교사/상태)
섹션 B: 최근 반 소식 카드 3개
섹션 C: 빠른 바로가기 카드 (급식 · 영양 안내, 서식 자료실, 1:1 문의 등)
```

### 2.2 Class News (`/parents/posts`, `/parents/posts/[id]`)
```
Hero: Class News 배지 + 설명 텍스트

List (테이블형 게시판)
┌ Table Header: No. / 반 / 제목 / 게시일 / 첨부 ─────────────────────┐
| 각 행: 번호(역순), 반 이름, 제목(링크), 2줄 요약                  |
| 첨부 칸: 사진/파일 개수 뱃지 (없음 시 회색 라벨)                   |
| 상단 토글: 최근 40건 안내 + 캘린더/사진/첨부 아이콘 레전드         |
└────────────────────────────────────────────────────────────────────┘

Detail (모달 + 라이트박스 스타일)
┌ Overlay: 반투명 배경 + 중앙 카드 ────────────────────────────────┐
| Header: Class News 배지, 게시일, 사진 개수, 반 이름, 닫기 버튼    |
| Summary · 본문 (pre-line, 파스텔 박스)                             |
| 사진 갤러리: 2~3열 카드, hover 시 딤처리 + 새 창 아이콘           |
| 첨부 자료: Paperclip 아이콘 + 외부 링크 리스트                    |
└────────────────────────────────────────────────────────────────────┘
```

### 2.3 Schedule (`/parents/schedule`)
```
Hero: Schedule 배지 + 안내 카피

Calendar Section
┌ Filter Row ──────────────────────────────────────┐
| [이전달] 월/연도 [다음달]  [이번달 보기]          |
| MealType 필터 버튼 (공지/체험/휴일 유사 컴포넌트) |
└──────────────────────────────────────────────────┘
┌ Monthly Calendar (6x7) ──────────────────────────┐
| 각 셀: 날짜 + 이벤트 배지(상태, 반, 공개 범위)    |
| 취소 일정은 붉은 배경 + 사유 텍스트              |
└──────────────────────────────────────────────────┘

List Section
┌ Card ────────────────────────────────────────────┐
| Classroom badge · Audience scope                 |
| Title + Date range + Location                    |
| Description                                      |
| Status badge (게시 / 승인 대기 / 취소)           |
| 취소 사유 (존재 시)                               |
└──────────────────────────────────────────────────┘
```

### 2.4 Meals & Nutrition (`/parents/meals`)
```
Hero: Nutrition 배지 + 설명 텍스트

Section A — Meal Calendar
┌ Controls: [이전달] [월 텍스트] [다음달] [이번달 보기] ─────────┐
| MealType 필터 버튼 (조식/중식/석식/간식/기타) + 전체 보기      |
└──────────────────────────────────────────────────────────────┘
┌ 6x7 Calendar Grid ───────────────────────────────────────────┐
| 각 셀: 날짜 번호 + “오늘” 배지 (해당일)                      |
| 이벤트칩: 색상 점 + 식단 유형 + 공개 범위 배지 + 메뉴 3줄    |
| Notes / 알레르기 요약                                       |
└──────────────────────────────────────────────────────────────┘

Section B — 다가오는 식단 (날짜 그룹 카드)
┌ Date Card ──────────────────────────────────────────────┐
| Heading: YYYY.MM.DD (요일)                               |
| Meal row: 유형 · 공개범위                                |
| 메뉴 리스트 (bullet)                                     |
| Notes / 알레르기                                         |
| 사진 갤러리 (최대 3개 프리뷰 + 잔여 개수)                 |
| 첨부 자료 링크 리스트                                    |
└──────────────────────────────────────────────────────────┘

Section C — 영양사 공지
┌ Bulletin Card ──────────────────────────────────────────┐
| Category badge · 게시일                                 |
| Title                                                   |
| Content (pre-line)                                      |
└──────────────────────────────────────────────────────────┘
```
> 식단 공개 범위는 `학부모/교직원/전체` 값을 지원하나, 현재 퍼블릭 웹에는 노출되지 않음.

### 2.5 Resources (`/parents/resources`)
```
Hero + 버튼
서식 자료 카드 리스트 / 운영위원회 자료 카드 리스트 (탭/필터)
```

### 2.6 1:1 Inquiries (`/parents/inquiries`)
```
왼쪽: 문의 작성 폼 (유형, 제목, 내용) + Toast
오른쪽: 최근 문의 카드 (상태 배지, 본문, 관리자 답변 버블)
```

## 3. Admin Console

### 3.1 Class Posts (`/admin/class-posts`)
```
Hero: Admin Console breadcrumb + CTA(대시보드)
Create Form: 반 선택 · 제목 · 요약 · 본문 · 첨부 URL 리스트 · 프런트 노출 가이드(테이블/모달 매핑)
승인 대기 패널 (관리자 전용): 카드 리스트 + [게시 승인] [숨김 처리]
전체 게시글 테이블:
| 반 | 제목/요약 | 상태 배지(승인 대기/게시/숨김) | 게시일 | 액션 (미리보기/수정/상태 전환/삭제) |
Teacher view: 담당 반 미배정 안내, 숨김 → “재검토 요청” 노출
```

### 3.2 Class Schedules (`/admin/class-schedules`)
```
Hero + 일정 생성 폼(반/전체, 유형, 공개 범위, 기간, 장소, 설명)
승인 대기 패널: 일정 카드 + [게시 승인] [취소 처리(사유 입력)]
등록된 일정 테이블:
| 대상 | 제목/설명 | 기간 | 상태(게시/승인 대기/취소) + 취소 사유 | 장소·유형·공개 범위 | 액션(게시/초안/취소/재게시/삭제) |
Teacher view: 담당 반 미배정 경고, 상태 안내 문구
```

### 3.3 Meals & Nutrition (`/admin/meals`)
```
Hero + 권한(관리자/영양사)
식단 생성 폼: 날짜, 유형, 메뉴, 메모, 알레르기, 공개 범위, 첨부(사진/파일), 프런트 노출 가이드(퍼블릭/학부모 카드 예시)
등록된 식단 테이블:
| 날짜 | 유형 | 메뉴 + 메모 + 공개 범위 | 알레르기 | 첨부 링크 | 액션(수정/삭제) |
영양 게시물: 작성 폼 + 카드 리스트 (카테고리, 게시 상태, 본문, 상태 전환)
```

### 3.4 기타
```
뉴스/자료실/문의 관리: 테이블 + 필터 + 상태/게시 제어 (기존 구상 유지)
역할 기반 내비게이션: 관리자 → 전체 메뉴, 교사 → 배정 반 콘텐츠, 영양사 → 식단/영양 메뉴
```
