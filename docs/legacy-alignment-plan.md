# Legacy Alignment Plan — Shinchon Montessori Revamp

## Context
- Legacy 사이트는 표 기반 UI와 팝업 중심 플로우(급식, 반별 활동 등)에 익숙한 학부모·교직원이 사용.
- 2025 리뉴얼은 현대식 카드/그리드/페이지 내 뷰를 채택하면서, 기존 사용자가 기대하는 위치·명칭·레퍼런스가 달라짐.
- 사용자 요구: “기존 틀을 크게 벗어나지 않은 채 낡은 UI만 개선”.

## Action Plan
1. **네비게이션 복구**
   - 상단 메뉴를 Legacy와 동일한 5개 축(유치원 소개 / 교육환경 / 교육과정 / 우리들 이야기 / 신촌소식)으로 재배치.
   - 각 메뉴의 하위 항목을 기존 URL/경로에 맞춰 연결하고, 학부모용 메뉴(반별 교육활동, 오늘의 식단 등)는 로그인 흐름과 연동.

2. **급식 UI (오늘의 식단)**
   - 월간 달력 표를 기본 레이아웃으로 유지하되, 셀 클릭 시 오른쪽 패널 또는 얕은 모달로 상세(사진, PDF, 알레르기) 표시.
   - 상단 탭(오늘의 식단 / 급식소통함), 월 이동 UI, 요일 헤더 등 기존 구조 유지 → 스타일과 접근성만 개선.

3. **반별 교육활동**
   - 목록 화면은 테이블(날짜·반·제목·첨부) 기반으로 복구, 썸네일은 아이콘/미리보기 스몰 이미지.
   - 상세는 Modal/Lightbox로 사진/본문/첨부를 보여줘 팝업 경험을 현대식으로 대체.

4. **로그인/권한 흐름**
   - 로그인 전 요약 정보, 로그인 모달을 통해 상세 페이지 접근.
   - 로그인 후에는 기존과 동일한 위치에서 자료 확인(예: 우리들 이야기 → 반별 교육활동).

5. **관리자 안내**
   - 관리자 폼(급식/반 소식)에 “프런트 출력 예시” 및 필드 설명 추가 → 기존 레이아웃에 맞춘 입력 가이드.
   - 기능 구조는 그대로 유지, 필요 시 뷰모델/응답 확장만 적용.

## Progress Log

- 2025-10-29 — Stage 3 완료: 급식 공개 범위(`전체/학부모`)를 Legacy 기대에 맞춰 정리하고 퍼블릭 `/meals` 페이지를 동일한 테이블/카드 UX로 제공.
